# Portfolio Personnel – ABOULFALAH Khadija

## Description

Ce projet est un **site web de portfolio personnel** développé en **HTML5 et CSS3**.  
Il présente mon profil académique, mes compétences techniques, mes expériences professionnelles et quelques projets réalisés dans le domaine de l’informatique et de la Data Science.

L’objectif de ce portfolio est de fournir une **présentation claire et professionnelle** de mon parcours en tant qu’étudiante en **Master 1 SDIA (Systèmes Décisionnels et Intelligence Artificielle)** à l’ENSET Mohammedia.

Le site est conçu pour être **simple, responsive et accessible**, tout en mettant en valeur mes compétences en **développement web et en technologies informatiques**.

# Objectifs du projet

- Créer un **portfolio professionnel en ligne**
